# DevAura — Discover Your Developer Energy

**DevAura** is a cinematic, interactive developer archetype experience. It analyzes your typing patterns, keywords, and cognitive style to reveal your **developer energy archetype**. A perfect blend of **dark cyber aesthetics, particle animations, and real-time interactivity**, built purely with HTML, CSS, and JavaScript.

---

## 🌌 Features

- **Interactive Typing Analysis**: Tracks typing rhythm, intensity, and input content in real time.  
- **Developer Archetype Scan**: Reveals your dominant archetype based on your input: Visionary, Catalyst, Creator, Purist, Builder, etc.  
- **Cinematic UI**: Full-screen dark mode with glitch effects, neon accents, and particle network background.  
- **Dynamic Stat Bars**: Vision, Execution, Creativity, Precision meters adjust based on your input.  
- **Responsive Design**: Works perfectly on all screen sizes — desktop, tablet, mobile.  
- **Re-run / Reset**: Re-analyze or reset at any time.  
- **Session Memory**: Stores last archetype in local storage for reference.  

---

## 🚀 How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).  
3. Start typing your thoughts in the textarea.  
4. Click **INITIATE ANALYSIS** to reveal your developer archetype.  
5. Use **RESET** to start over.

---

## 🛠 Tech Stack

- HTML5  
- CSS3 (with flexbox, animations, gradients, and responsive layout)  
- Vanilla JavaScript (ES6)  
- Canvas API for particle network background  

No frameworks or external JS libraries required. Fully lightweight and self-contained.
